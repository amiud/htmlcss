Tags

Tags das imagens

Use a tag picture para as imagens dinâmicas. A tag media juntamente com min ou max para o tamanho, type para o tipo de mídia, e a tag srcset para setar (set) qual a imagem e onde ela fica (source).

Tags do audio

Use a tag audio para áudios dinâmicos. A tag scr para dizer onde se localiza o áudio (source), e a tag type para o tipo de mídia.
Também dentro da chave audio tem a tag preload para dizer o quanto do arquivo será baixado em primeira mão, antes de ele ser tocado. A tag controls para adicionar um reprodutor, a tag loop para dar loop no áudio e a tag autoplay (não sei para que serve ainda).
