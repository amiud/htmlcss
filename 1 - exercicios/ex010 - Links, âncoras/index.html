Link externo

Usando a tag a juntamente com o target="_blank", você pode criar uma âncora para um link externo. E usar com rel="external", o mecanismo de busca irá vasculhar o site, já que faz parte do mesmo. Veja os exemplos:

Você pode acessar meu repositório público no GitHub.
Você também pode acessar meu canal no youtube.

Usando links internos

Usando a tag a juntamente com o target="_self", você pode criar uma âncora para um link interno. Para avançar deve-se usar a tag a com rel="next", e para voltar deve-se usar a tag a juntamente com o rel="prev".

Esta é a primeira página do meu site, se você quiser também pode acessar a minha segunda página
Você também pode acessar a nossa página de downloads
