Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito, usando a tag b (não semântica).

Nesta frase, temos um termo em destaque, usando a tag strong (semântica).

Itálico / Ênfase

Nesta frase,temos um termo em itálico usando a tag i (não semântica).

Nesta frase, temos um termo em ênfase usando a tag em (semântica).

Texto marcado

Podemos também, criar um texto marcado usando a tag mark (semântica).

E também, podemos adicionar cor no marcador com o CSS, desta forma (semântica com forma).

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo (semântica, big se tornou obsoleta, então deve aumentar as letras usando CSS).

Texto excluído

Podemos marcar um texto como excluído para indicar de ele deve ser lido mas não considerado, usando a tag del (semântica).

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag ins (semântica).

Também existe o sublinhado, usando a tag u (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag sup (semântica).

texto subscrito

Para inserir coisas do tipo H2O, usando a tag sub (semântica).

Dicas

CTRL + SHIFT + P, com a palavra/frase desejada selecionada, wrap with abbreviation, para colocar a palavra/frase entre uma tag.
