Listas ordenadas

Para criar listas ordenadas (listas em que a ordem importa), use a tag ol. E pode-se usar a tag type para mudar o parâmetro da lista, e a tag start para definir qual vai ser o início.

Lista de definições

Usando a tag dl se cria uma lista de definições, dentro dela com a tag dt você cria o termo, e finalmente com a tag dd você dá a definição deste termo.

HTML
Linguagem de marcação para a criação do conteúdo de um site.
